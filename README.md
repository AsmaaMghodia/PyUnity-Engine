# GoEngine
2D OpenGL Scripting Game Engine.
  - Simple
  - Quick
  - Efficient

# Features:

  - GameObjects Transformation.
  - Camera Transformation.
  - Collision Detection.
  - Sprite Rendering.
  - lighting.
  - Timing.
  - ...
 

# Notes:
    This game engine is pure OpenGL EXCEPT for the usage of pyGame to load images.
